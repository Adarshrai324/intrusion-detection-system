# 🛡️ AI Driven Intrusion Detection System

## 📌 Overview
This project is an AI-based Intrusion Detection System developed using Machine Learning, Deep Learning, and Streamlit. The system analyzes network traffic and predicts whether the traffic is normal or malicious.

---

## 🚀 Features
- Real-time intrusion detection
- AI-based attack prediction
- Streamlit web interface
- Machine learning model integration
- User-friendly deployment

---

## 🧠 Technologies Used
- Python
- Streamlit
- NumPy
- Pandas
- Scikit-learn
- TensorFlow
- Pickle

---

## 📂 Dataset
We used the NSL-KDD dataset for training and testing the intrusion detection model.

---

## ⚙️ Project Workflow

1. Data Collection  
2. Data Preprocessing  
3. Feature Selection  
4. Model Training  
5. Model Evaluation  
6. Model Deployment using Streamlit  
7. Real-time Prediction  

---
